# PlacePro – Placement Management System

A full-stack campus placement portal that connects students, recruiters, and admins in one workflow.

## Features

- Student registration, login, and skill management
- Job discovery with filters, skill match, and placement score
- One-click apply and application tracking
- Recruiter job posting and applicant management
- Status updates, shortlist workflow, and messaging
- Admin analytics dashboard and reports

## Tech Stack

- Backend: Python + Flask
- Database: MySQL
- Frontend: HTML (Jinja templates) + CSS

## Roles

- **Student:** apply for jobs, view status, chat with recruiters
- **Recruiter:** post jobs, review applicants, update status
- **Admin:** monitor metrics and system-wide stats

## Quick Start

1. Create the database and tables in MySQL.
2. Update database credentials in `app.py`.
3. Run the server:

```bash
python app.py
```

## Database Setup

Use your SQL script to create schema and seed data (Skills, Students, Companies, Jobs, etc.).
